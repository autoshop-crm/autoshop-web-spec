# Дипломный проект: CRM для автосервиса

## Что я хочу сделать

Вообщем, я бы хотел сделать CRM для автосервиса. Он будет юзать стороннее API для сверки стоимости автозапчастей.

Будет инфа по всем клиентам.

Будет web-панель для владельцев бизнеса, автомехаников и менеджеров.

## Микросервисы

Если делить на микросервисы, то что-то типа:

### 1. Сервис для клиентов
Все с клиентами, их поиск, фильтрация ну и CRUD соответственно.

### 2. Управление авто
Весь каталог авто, которые сейчас находятся на ремонте, и их история.

### 3. Управление заказами
Все для механиков: какие сейчас есть заказы, что нужно сделать, статусы работы.

### 4. Работа с закупкой запчастей
Здесь интеграция с сторонним API.

### 5. Сервис уведомлений
Возможно что-то типа уведомлений через Spring.

### 6. Сервис авторизации
Там будет все для авторизации.

### 7. Хранилище файлов
Можно также сделать хранилище файлов для документов машины. Как раз познакомлюсь с NoSQL, S3.
